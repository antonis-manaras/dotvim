# dotvim
Vim plugins configuration

## Installation:

* Move ~/.vimrc under .vim (note that dot (.) is removed, vimrc will not be hidden under .vim): 
	
	```
	mv ~/.vimrc ~/.vim/vimrc
	```
* Create symlinks:
	
	```
	ln -s ~/.vim/vimrc ~/.vimrc
	```
Install every plugin under: 
	
	```
	.vim/bundle/
	```
usinge git submodules.
